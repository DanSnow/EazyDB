EazyDB
======

**Don't use this in production.**  
This is an eazy db which write in Crystal.  

## Implement ##

- CURD
- Offset index
- Schema
  - str (varchar)
  - num (int32)

## Built with ##
- [commander](https://github.com/mrrooijen/commander)
- [cyrstal-binary_parser](https://github.com/DanSnow/crystal-binary_parser)
